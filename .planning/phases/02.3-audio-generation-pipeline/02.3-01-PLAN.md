    **audio_generator.py — Create AudioGenerator class mirroring ImageGenerator pattern:**

    Create `spanish-content-pipeline/pipeline/audio_generator.py`:

    ```python
    """Pass 6: Generate sentence audio via TTS APIs (Google Cloud TTS / OpenAI)."""
    ```

    Class `AudioGenerator`:
    - Constructor: `__init__(self, config: DeckConfig, api_key: str, output_base: Path | None = None, max_retries: int = 3)`
      - Store config, api_key, output_base (default Path("output")), max_retries
      - Read audio_generation config from config
      - Initialize provider-specific client lazily (only when needed, to allow mocking in tests)
    - `_deck_dir(self) -> Path` — returns `output_base / config.deck.id` (same as ImageGenerator)
    - `_sentence_key(self, chapter: int, sentence_index: int) -> str` — returns `f"ch{chapter:02d}_s{sentence_index:02d}"` (same format as images)
    - `_content_hash(self, text: str) -> str` — returns `hashlib.sha256(text.encode('utf-8')).hexdigest()[:16]`

    Provider methods:
    - `_generate_google(self, text: str) -> bytes` — uses `google.cloud.texttospeech` SDK:
      - `SynthesisInput(text=text)`
      - `VoiceSelectionParams(language_code=config.languages.target_code, ssml_gender=SsmlVoiceGender.MALE)` — read gender from `config.audio_generation.voice_gender` mapping ("male" -> MALE, "female" -> FEMALE)
      - `AudioConfig(audio_encoding=AudioEncoding.MP3, speaking_rate=config.audio_generation.speaking_rate)`
      - Return `response.audio_content`
    - `_generate_openai(self, text: str) -> bytes` — uses `openai.OpenAI` SDK:
      - `client.audio.speech.create(model="gpt-4o-mini-tts", voice="alloy", input=text, response_format="mp3")`
      - Collect bytes via `response.iter_bytes()` and return

    Core methods:
    - `generate_sentence_audio(self, sentence: SentencePair) -> AudioManifestEntry`:
      - Build key and rel_path (`audio/{key}.mp3`)
      - Try calling provider method, write bytes to file
      - Return success entry with content_hash, or failed entry with error
      - Retry logic: wrap API call in retry loop (same pattern as ImageGenerator — retry on 5xx, backoff with `time.sleep(2 * (attempt + 1))`)
    - `generate_all(self, sentences: list[SentencePair]) -> AudioManifest`:
      - Load existing manifest if present (for resumability)
      - For each sentence: compute content_hash, skip if unchanged in manifest, otherwise generate
      - Write updated manifest to `audio_manifest.json`
      - Return AudioManifest

    IMPORTANT: Read language from `config.languages.target_code`, NOT hardcoded "es-ES". For Google TTS the language_code format may need to be adapted (e.g. "es" -> "es-ES" or "es-US"). Use the config value directly — Google Cloud TTS accepts short codes like "es" and selects automatically.
