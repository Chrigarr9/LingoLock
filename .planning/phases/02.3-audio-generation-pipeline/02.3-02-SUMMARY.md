## Decisions Made
- **Pass 6 uses lazy import:** AudioGenerator imported inside the block (after API key check) to avoid ImportError when google-cloud-texttospeech not installed
- **Audio manifest pattern mirrors images:** Consistent structure (manifest.json → copy files → generate require() map) for maintainability
- **cardAudios map uses static require():** Metro bundler compatibility requires compile-time requires (not runtime fs.readFileSync)
- **Bundled assets first, URI fallback:** Enables both pipeline-generated MP3s and future HTTP URLs
- **Empty cardAudios export added:** Temporary fix to unblock TypeScript compilation until next build-content.ts run with real audio
