### Next Steps

**To use audio generation:**

1. Install TTS dependencies:
   ```bash
   cd spanish-content-pipeline
   pip install google-cloud-texttospeech>=2.16.3 openai>=1.0.0
   ```

2. Add TTS API key to .env:
   ```bash
   # For Google Cloud TTS (primary):
   GOOGLE_TTS_API_KEY=your-key-here
   
   # OR for OpenAI TTS (secondary):
   OPENAI_API_KEY=your-key-here
   ```

3. Run pipeline with audio generation:
   ```bash
   python scripts/run_all.py --config configs/spanish_buenos_aires.yaml
   ```

4. Build content bundle:
   ```bash
   cd ..
   npx tsx scripts/build-content.ts
   ```

5. Audio now available in ClozeCard when `card.audio` is populated

**To skip audio generation:**

Option 1: Don't set API key (graceful skip with WARNING)
Option 2: Pass `--skip-audio` flag to run_all.py
Option 3: Set `audio_generation.enabled: false` in config YAML
