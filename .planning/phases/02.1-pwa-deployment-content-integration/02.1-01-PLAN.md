<objective>
Create the web platform storage adapter that mirrors the native MMKV storage API using localStorage.

Purpose: This is the critical-path prerequisite for the web build. The existing `storage.ts` imports `react-native-mmkv` which is a native C++ module that crashes on web. Metro bundler automatically resolves `.web.ts` over `.ts` for web builds, so creating `storage.web.ts` with identical exports lets all service layer code (`statsService.ts`, `fsrs.ts`, `cardSelector.ts`) work unchanged on web.

Output: `src/services/storage.web.ts` — drop-in localStorage replacement for MMKV
</objective>
