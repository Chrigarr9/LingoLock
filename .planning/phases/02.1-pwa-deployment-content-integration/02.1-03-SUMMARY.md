## User Setup Required

**Task 3 (Vercel deployment) requires manual configuration:**

1. Go to https://vercel.com → "Add New..." → "Project"
2. Import the LingoLock GitHub repository
3. Configure build settings:
   - Framework Preset: **Other**
   - Build Command: `npm run build:web`
   - Output Directory: `dist`
   - Install Command: `npm install` (default)
4. Click "Deploy" and wait for the first build to complete
5. Verify the live Vercel URL loads the app and practice session works
6. Confirm auto-deploy: push a trivial change to master, verify Vercel rebuilds
7. Confirm preview deploys: create a test PR, verify Vercel generates a preview URL

Once complete, requirements `PWA-VERCEL-DEPLOY` will be satisfied.
