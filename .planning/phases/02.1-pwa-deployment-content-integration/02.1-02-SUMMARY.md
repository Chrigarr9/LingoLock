## Decisions Made

- Used `#FFA056` brand orange as themeColor (browser UI tinting when installed)
- Used `#fffcf2` warm background as backgroundColor (splash screen color while loading)
- `display: standalone` chosen over `fullscreen` to keep OS-level back gesture accessible
- Service worker cache version string `lingolock-v1`: bump to `v2` to force full cache invalidation on breaking deploys
- Navigation requests are network-first so a Vercel deploy immediately serves new HTML (not stale cached HTML)
- Static assets are cache-first because Metro appends content hashes to JS/CSS bundle filenames — a new deploy means new URLs, so the old cached files are never stale
