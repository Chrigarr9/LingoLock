  <action>
Create `scripts/build-content.ts` — a Node.js TypeScript script that reads pipeline output and generates `src/content/bundle.ts`.

**Input files:**
- `spanish-content-pipeline/output/es-de-buenos-aires/words/chapter_XX.json` — per-chapter sentences + word annotations
- `spanish-content-pipeline/output/es-de-buenos-aires/translations/chapter_XX.json` — translations (same data as sentences in words file)
- `spanish-content-pipeline/output/es-de-buenos-aires/vocabulary.json` — global vocabulary with frequency_rank and cefr_level

**Processing logic:**

1. **Discover chapters:** Glob `words/chapter_*.json` files, sort by chapter number.

2. **Load vocabulary.json** for frequency_rank, cefr_level, and distractor pool.

3. **For each chapter, for each word annotation:**
   - Find the sentence (by sentence_index) the word belongs to. A word belongs to the sentence where `word.source.toLowerCase()` appears in `sentence.source.toLowerCase()`. Use the first matching sentence from the chapter.
   - Create cloze sentence: Replace the FIRST occurrence of `wordInContext` (the word's `source` field) in the sentence's `source` with `"_____"`. Use a case-insensitive word-boundary-aware replacement. If no occurrence found, log a warning and skip this card.
   - Get sentence translation from the matching sentence's `target` field.
   - Look up `cefr_level` from vocabulary.json by matching on the word's `lemma.toLowerCase()`.
   - Generate card ID: `"{lemma}-ch{chapter:02d}-s{sentenceIndex:02d}"` (e.g., "habitación-ch01-s01").

4. **Generate distractors** for each card:
   - From vocabulary.json, find entries with same `pos`.
   - If CEFR level available, prefer entries within ±1 CEFR level rank (A1=1, A2=2, B1=3, B2=4, C1=5, C2=6).
   - Exclude the correct lemma.
   - Pick 3 random distractors (for MC4; MC2 will use 1 at runtime). Use the entry's `source` field (the lemma).
   - If fewer than 3 same-POS distractors available, fill with any POS.
   - Deduplicate — no distractor should equal the correct word.

5. **Deduplicate cards:** If same lemma appears multiple times in same sentence, keep only first. If same lemma appears in multiple sentences of the same chapter, keep all (different contexts).

6. **Write output** to `src/content/bundle.ts`:
```typescript
// AUTO-GENERATED by scripts/build-content.ts — DO NOT EDIT
// Generated: {ISO date}
// Source: spanish-content-pipeline/output/es-de-buenos-aires/
import type { ClozeCard, ChapterData } from '../types/vocabulary';

export const CHAPTERS: ChapterData[] = [
  {
    chapterNumber: 1,
    cards: [
      { id: "habitación-ch01-s01", lemma: "habitación", ... },
      ...
    ]
  },
  ...
];

/** Flat array of all cards across chapters */
export const ALL_CARDS: ClozeCard[] = CHAPTERS.flatMap(ch => ch.cards);

/** Look up card by ID */
export function getCardById(id: string): ClozeCard | undefined {
  return ALL_CARDS.find(c => c.id === id);
}

/** Get cards for a specific chapter */
export function getChapterCards(chapterNumber: number): ClozeCard[] {
  return CHAPTERS.find(ch => ch.chapterNumber === chapterNumber)?.cards ?? [];
}

/** Total card count */
export function getTotalCards(): number {
  return ALL_CARDS.length;
}
```
